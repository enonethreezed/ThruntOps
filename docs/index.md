# ThruntOps

A Ludus-based lab environment for TTP testing and security research.

Deployed on Proxmox via [Ludus](https://docs.ludus.cloud). Dual Active Directory domain, ADCS, IIS + MSSQL, GitLab CE, and Elastic SIEM on VLAN 50 (`10.2.50.0/24`).
